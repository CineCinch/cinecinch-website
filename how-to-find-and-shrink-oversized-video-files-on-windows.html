How CineCinch finds the real space hogs

Sorting by file size misses short but inefficient files. CineCinch calculates MB per second and ranks files so you see which ones will free the most space when compressed.

Quick workflow

  • Scan a folder and get a ranked list by MB/sec inefficiency.
  • Select the worst offenders and pick a preset (Plex/Jellyfin friendly).
  • Queue and run batch compression locally; pause/resume as needed.
